# 100 Dólares

## 1. Introdução
Dado que são elicitados vários requisitos, para saber por onde começar o desenvolvimento, torna-se necessário priorizar esses requisitos. (Milene Serrano, Requisitos - Aula 07, página 30).

Dessa forma, nessa seção a técnica utilizada para a priorização dos requisitos é a 100 doláres.

## 2. Metodologia
Essa técnica consiste em, basicamente, pedir para o usuário/cliente distribuir 100 doláres entre todos os requisitos disponíveis. Assim, espera-se que os requisitos mais vitais e importantes receberão um maior valor, enquanto os menos urgentes terão um menor "investimento" recebido pelo usuário.

Com essa técnica, encontra-se, de forma simples e intuitiva, os requisitos mais valiosos na visão do usuário que usará o aplicativo.
